Contagem de estoque

Leia Antes de Usar

Cuidado ao atualizar a página! Este software foi desenvolvido para auxiliar na contagem de estoque, facilitando o dia a dia e garantindo um controle absoluto do estoque. Existem 3 funções principais:

  • Calculadora de Quantidade Total: Este botão abre uma calculadora para calcular o total de itens empilhados em uma área (por exemplo, paletes ou caixas), com base na altura x largura.
  • Adicionar Produto: Permite adicionar os itens que você já contou no estoque.
  • Remover Produto: Use esta função caso tenha adicionado um item incorretamente.
  • Gerar Relatório: Copie a lista formatada para colar no WhatsApp com a contagem final.

Importante: Evite atualizar a página, pois os dados podem ser perdidos.
